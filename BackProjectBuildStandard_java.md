# 3.开发心得

## 3.1.大事化小，切勿在一个方法中完成全部逻辑的书写，以便于后期维护

示例：分页+下钻逻辑（工贸-网格-服务商-服务兵）+时间

可以优先获取分页，然后每一个分页项，以实现一个分页项为单独方法，使用多线程去处理提高接口速度
